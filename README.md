## API Documentation

### Get Started

To get started, run `npm start` from the root directory.

The server will run on `http://localhost:5000/`.


> `POST /flight/create`

URL: `POST http://localhost:5000/video`

##### Request:

| Field Name | Required | Description |
| ---------- | -------- | ----------- |
| Header     | Title    | Hi          |
| Paragraph  | Text     | Ok          |
